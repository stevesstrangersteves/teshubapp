## 🆘 Hỗ Trợ Từng Loại Vấn Đề

### 🐛 Báo Cáo Lỗi (Bug Report)

**Thông Tin Cần Cung Cấp**:
```
1. Describe Bug (Mô tả lỗi):
   - "Khi tôi làm X, ứng dụng Y"

2. Steps to Reproduce (Tái hiện):
   - Bước 1: ...
   - Bước 2: ...
   - Bước 3: ...
   → Lỗi xảy ra

3. Expected vs Actual (So sánh):
   - Kỳ vọng: ...
   - Thực tế: ...

4. Environment (Môi trường):
   - OS: [Windows/macOS/Linux] + Version
   - App Version: [1.0.0]
   - API Version: [ElevenLabs SDK version]

5. Attachments (Tệp đính kèm):
   - Screenshot / Log file
   - Video (nếu cần)
```

**Ví Dụ Báo Lỗi Tốt**:
```
[BUG] TTS fails with Vietnamese characters over 500 chars

Steps to Reproduce:
1. Open TTS tab
2. Enter Vietnamese text: "Xin chào thế giới..." (600 chars)
3. Select voice: Rachel
4. Click Generate
→ Error: "Invalid character encoding"

Expected: Audio file generated
Actual: Error dialog shown

OS: Windows 11 Build 22H2
App: v1.0.0
API Key: Valid (tested with shorter text)
```
