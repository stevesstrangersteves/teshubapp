# TeSHub Desktop - Giới Thiệu

**TeSHub Desktop** là ứng dụng desktop chuyên nghiệp được xây dựng trên Electron, cung cấp các dịch vụ AI voice xác thực từ ElevenLabs. Ứng dụng cho phép bạn tạo, xử lý và quản lý âm thanh AI mạnh mẽ trực tiếp từ máy tính của mình.

## Giới Thiệu

TeSHub Desktop mang lại sức mạnh của các công cụ AI voice chuyên nghiệp vào tầm tay của bạn. Dù bạn là nhà sáng tạo nội dung, nhà phát triển ứng dụng, hay người tạo podcast, ứng dụng này cung cấp tất cả những công cụ bạn cần.

### Tính Năng Chính

- **Text-to-Speech (TTS)**: Chuyển đổi văn bản thành giọng nói tự nhiên chỉ trong vài cú nhấp
- **Speech-to-Text (STT)**: Phiên âm âm thanh thành văn bản với độ chính xác cao
- **Xử Lý Song Song**: Xử lý hàng trăm TTS requests cùng lúc
- **Tạo Âm Thanh**: Tạo hiệu ứng âm thanh từ mô tả văn bản
- **Soạn Nhạc**: Tạo bản nhạc gốc với AI
- **Quản Lý Key Pool**: Quản lý nhiều API keys một cách hiệu quả

## Hệ Thống Yêu Cầu

### Tối Thiểu
- **Windows**: Windows 10 hoặc mới hơn
- **macOS**: macOS 12 (Monterey) hoặc mới hơn
- **Linux**: Ubuntu 20.04 LTS hoặc tương đương
- **RAM**: 4GB (8GB khuyên cáo)
- **Ổ cứng**: 2GB khoảng trống

### Khuyên Cáo
- **RAM**: 8GB trở lên cho xử lý song song
- **CPU**: Processor đa nhân (4+ cores)
- **Internet**: Kết nối 10Mbps trở lên cho upload/download audio

## Tính Năng Nổi Bật

### ✨ Text-to-Speech (TTS)
Chuyển bất kỳ văn bản nào thành giọng nói tự nhiên. Chọn từ hàng chục giọng AI, điều chỉnh tốc độ, điểm nhấn và phong cách.

```
Ví dụ: "Xin chào thế giới" → 🔊 File MP3 chất lượng cao
```

### 🔄 Xử Lý Song Song (Parallel TTS)
Xử lý hàng loạt text cùng một lúc. Lý tưởng cho podcast, ebooks và sản xuất video.

```
10 đoạn văn → 🚀 Hoàn thành trong vài phút
```

### 🎙️ Speech-to-Text (STT)
Phiên âm audio thành văn bản chính xác. Hỗ trợ nhiều ngôn ngữ và định dạng file.

### 🔊 Hiệu Ứng Âm Thanh
Tạo SFX tùy chỉnh từ mô tả. Không cần kinh nghiệm audio.

### 🎵 Soạn Nhạc
Tạo bản nhạc gốc và bài hát với lời bằng AI.

### 🔐 Quản Lý Key Pool
Quản lý nhiều ElevenLabs API keys:
- Theo dõi sử dụng từng key
- Tự động xoay chuyển keys
- Nhập keys từ CSV/Excel
- Mã hóa bảo mật AES-256
